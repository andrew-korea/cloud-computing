README

FOLDER DESCRIPTIONS

01 MongoDB connector
Establishes connection to MongoDB and handles .json file

02 Data preprocess
This folder contains code that preprocesses the data such that meaningful attributes are selected

03 Data Analysis
This folder performs data analysis on the preprocessed data

04 MongoDB pipeline
Contains code that can connect to MongoDB via Spark and performs join operations on Dataframe.

05 Final Code
cluster_program.py is the source code run on the cloud cluster than loads the data from MongoDB and fits a random forest model.
